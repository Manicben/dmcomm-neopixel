
LIST OF CODES FOR VARIOUS TOYS

Paste a code into the serial monitor, or use it as a command-line argument to dmscope.

Codes written as "V1" or "X1" must be "1". Codes written as "V2" or "X2" can be changed to "1" with no problem (choose whichever you prefer).

Note that all codes have an inverted meaning from what you see when using them. If the code says "I win the battle", the real toy loses the battle. If the code says "I give item A", the real toy receives item A. If the code reports a jogress evolution, the real toy will do the corresponding evolution if compatible.
